# M5.1：鱼骨画布视图底座

## 目标

将鱼骨主图从表格化时间线改造为坐标化画布。日期、主线和任务节点不再依赖 grid 单元格，而是位于同一块可平移、可缩放的 canvas stage 上。

## 范围

- 新增 `FishboneCanvasViewport`，记录 `panX`、`panY`、`canvasZoom`、`timeScale`、`centerDate`、`focusedLaneId` 和 `laneZooms`。
- 新增画布坐标换算：日期转 x 坐标，主线转 y 坐标，任务转 x/y 节点。
- 新增 `fishboneCanvasLayout.ts`，输出 date ticks、mainline spines 和 task nodes。
- 主渲染入口切换为 `fishbone-canvas-viewport` 和 `fishbone-canvas-stage`。
- 日期轴作为坐标层渲染，不再是 grid 列。
- 主线作为横向 spine 渲染，不再是表格行。
- 任务节点以绝对坐标挂到 `date + mainline` 对应点。
- 鼠标按住画布空白区域可平移画布。
- `Ctrl + 滚轮` 在主线外缩放整体画布。
- `Ctrl + 滚轮` 在某条主线区域内缩放单条主线。
- `Alt + 滚轮` 或在日期轴上 `Ctrl + 滚轮` 调整时间轴密度。
- 保留 M5 主线创建、修改、删除、排序、任务完成、任务主线选择和点击打开任务。

## 不做

- 不做 relation 曲线。
- 不做任务拖拽改日期。
- 不做任务拖拽改主线。
- 不做主线折叠、隐藏、筛选。
- 不做右侧工作台。
- 不做底部看板。
- 不做 task-index 同步。
- 不创建默认主线。
- 不覆盖已有 `PlanningSystem`。

## Obsidian 验收

- 主图不再呈现为表格 grid。
- 日期轴、主线脊骨和任务节点在同一个画布中。
- 按住画布空白处拖动可以平移。
- `Ctrl + 滚轮` 在主线外缩放整体画布。
- `Ctrl + 滚轮` 在主线上只缩放该主线。
- `Alt + 滚轮` 或在日期轴上 `Ctrl + 滚轮` 改变日期间距。
- 任务节点随画布平移和缩放移动。
- 日期轴随时间密度变化重新排布。
- checkbox、select、任务点击打开不会触发画布拖动。
- M5 的主线管理能力不回退。
