# M6.5：L 型模块容器与下方状态工作台

## 目标

把右侧辅助模块和下方状态工作台统一为一个 L 型模块容器区域。右侧继续承载今日聚焦、本周重点和进度类模块；下方新增待办、进行中、已完成三列状态工作台；快速笔记 / Codex 输入以悬浮输入框形式贴在下方工作台上方。

## 实现范围

### 1. L 型模块容器

- 鱼骨画布外层新增 `fishbone-dock-main`，上半区为画布 + 右侧模块。
- 下方新增 `fishbone-workbench-panel`，与右侧模块属于同一个工作台开关控制范围。
- 工具栏按钮从 `显示辅助 / 隐藏辅助` 调整为 `显示工作台 / 隐藏工作台`。
- 右侧面板宽度、右侧模块顺序和高度继续持久化。
- 下方工作台高度和列顺序持久化。

### 2. 下方三列状态工作台

- 下方三列为 `待办`、`进行中`、`已完成`。
- `todo / inbox` 进入待办。
- `doing / blocked` 进入进行中，阻塞任务显示阻塞标记。
- `done` 进入已完成。
- `canceled` 默认不进入状态工作台。
- 每列内部独立滚动，外层不整体滚动。
- 点击任务打开对应任务 md。
- 勾选任务写回 `done`，取消勾选写回 `todo`。

### 3. 拖拽写回

- 任务可拖拽到三列之间。
- 拖到待办写回 `status: todo`。
- 拖到进行中写回 `status: doing`。
- 拖到已完成写回 `status: done`。
- 写回后刷新鱼骨画布、右侧模块和下方工作台。
- 下方三列标题可横向拖拽调整顺序。

### 4. 悬浮快速输入骨架

- 快速输入框悬浮在下方工作台上方。
- 输入自然语言后生成候选任务预览。
- 候选任务展示标题、日期、主线、优先级和状态。
- M6.5 只做候选预览，不写入标准任务 md。
- 标准任务写入、自然语言拆解和 Codex 智能解析进入 M6.6。

## 不做

- 不做右侧模块和下方模块跨区域自由移动。
- 不做完整自然语言任务解析。
- 不做每日总结生成。
- 不做真实天气 API。
- 不修改任务 frontmatter schema。
