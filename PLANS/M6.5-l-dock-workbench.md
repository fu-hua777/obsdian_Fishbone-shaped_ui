# M6.5：L 型模块容器与下方状态工作台

## 目标

把右侧辅助模块和下方状态工作台统一为一个 L 型工作区域，但二者保持清晰独立：右侧面板独占右列并贯穿画布和下方工作台高度；下方工作台只位于画布下方，不延伸到右侧面板下面。快速笔记 / Codex 输入不属于模块容器，而是固定悬浮在鱼骨画布底部居中。

## 实现范围

### 1. L 型模块容器

- 鱼骨画布外层使用两列两行网格：左上为画布、左下为下方工作台、右侧为辅助面板。
- 右侧辅助面板跨越上下两行，避免下方工作台盖到右侧区域。
- 下方新增 `fishbone-workbench-panel`，只占据画布下方区域。
- 工作台开关同时控制右侧辅助面板和下方工作台。
- 工具栏按钮从 `显示辅助 / 隐藏辅助` 调整为 `显示工作台 / 隐藏工作台`。
- 右侧面板宽度、右侧模块顺序和高度继续持久化。
- 下方工作台高度和列顺序持久化。
- 右侧不渲染额外的“辅助面板”头块，直接显示功能模块。
- 今日进度和本周进度整合为一个 `进度概览` 模块，用两条横向进度条展示。
- 主线进度改为圆形进度环，接近概念图中的进度概览视觉。

### 2. 下方三列状态工作台

- 下方三列为 `待办`、`进行中`、`已完成`。
- `todo / inbox` 进入待办。
- `doing / blocked` 进入进行中，阻塞任务显示阻塞标记。
- `done` 进入已完成。
- `canceled` 默认不进入状态工作台。
- 每列内部独立滚动，外层不整体滚动。
- 点击任务打开对应任务 md。
- 勾选任务写回 `done`，取消勾选写回 `todo`。
- 下方任务条保持紧凑：左侧主线色图标，右侧主线色标签，中间显示任务标题、日期和优先级。

### 3. 拖拽写回

- 任务可拖拽到三列之间。
- 拖到待办写回 `status: todo`。
- 拖到进行中写回 `status: doing`。
- 拖到已完成写回 `status: done`。
- 写回后刷新鱼骨画布、右侧模块和下方工作台。
- 下方三列标题可横向拖拽调整顺序。
- 右侧模块标题和下方列标题显示拖拽手柄，用于调整模块或列的位置。

### 4. 悬浮快速输入骨架

- 快速输入框固定悬浮在鱼骨画布底部居中，不进入右侧或下方模块容器。
- 输入自然语言后生成候选任务预览。
- 候选任务展示标题、日期、主线、优先级和状态。
- M6.5 只做候选预览，不写入标准任务 md。
- 标准任务写入、自然语言拆解和 Codex 智能解析进入 M6.6。

## 不做

- 不做右侧模块和下方模块跨区域自由移动。
- 不做完整自然语言任务解析。
- 不做每日总结生成。
- 不做真实天气 API。
- 不修改任务 frontmatter schema。
