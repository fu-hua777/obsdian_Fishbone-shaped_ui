# M6：概念图差距与下方 UI 模块补足计划

## 当前已具备

- 鱼骨画布主体：用户主线、日期轴、任务节点、分支主线、关系线、画布缩放和平移。
- 右侧辅助区：今日聚焦、本周重点、今日/本周进度、主线进度等模块化容器。
- 任务状态写回：今日聚焦中可勾选完成和快速切换状态。
- 模块布局：右侧模块可拖拽排序，可长按模块下边缘上下调节高度，布局状态持久化。

## 与概念图相比仍缺的核心功能

### 1. 下方三列工作台

概念图底部有三个横向模块：

- 待办
- 进行中
- 已完成

当前右侧已取消状态速览，避免重复；后续应把状态工作流放到下方工作台。

建议实现：

- 下方工作台与鱼骨画布同屏，默认显示在画布下方。
- 三列各自内部滚动，外层不整体滚动。
- 支持任务点击打开。
- 支持任务状态在三列之间拖拽移动。
- 支持 `+ 添加任务` 入口，但写入仍走标准任务 md。

### 2. 快速笔记 / Codex 输入

概念图右侧有快速输入模块。

当前尚未实现。

建议实现：

- 先做输入框和候选任务预览。
- 候选任务必须显示标题、日期、主线、优先级、relations 和待确认项。
- 用户确认后才写入标准任务 md。
- 后续再接 planning-task-skill / Codex 自动解析。

### 3. 每日总结

概念图右侧底部有每日总结模块。

当前尚未实现。

建议实现：

- 显示今日总结状态：未生成 / 已生成。
- 生成前列出将使用的数据：今日完成、新增、进行中、阻塞、快速输入。
- 用户确认后写入 `PlanningSystem/DailyReports/YYYY-MM-DD_每日总结.md`。
- 已存在总结时提供打开、追加、覆盖三种明确动作。

### 4. 时间 / 天气

概念图右侧有时间天气模块。

当前尚未实现。

建议实现：

- 时间本地生成即可。
- 天气先做手动城市配置和占位信息；真实天气 API 后置。
- 不把天气作为 M6 核心阻塞项。

### 5. 顶部全局导航与工具细节

概念图顶部还有 Obsidian 风格全局入口、搜索、日历、关系、设置等图标。

当前已有鱼骨工具栏，但还不完整。

建议实现：

- 保留当前工具栏，逐步图标化。
- 加入“跳转日期”“关系显示”“辅助面板”“下方工作台”统一开关。
- 不在 M6 阶段做全局 Obsidian 外壳复刻。
