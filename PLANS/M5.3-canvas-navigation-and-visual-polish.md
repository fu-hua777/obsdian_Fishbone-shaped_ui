# M5.3：鱼骨画布导航、时间轴模式与概念图视觉打磨

## 目标

在 M5.1 坐标化画布和 M5.2 任务交互 / relation 关系线的基础上，把鱼骨主视图推进到更接近概念图的可用形态：顶部工具栏更紧凑、时间轴可切换模式、主线可折叠/隐藏/固定、任务密集时可聚合，并持久化用户的画布视图状态。

本阶段仍只处理鱼骨图主体，不实现右侧工作台。

## 范围

### 1. 顶部导航工具栏

- 增加 `今天`、`跳转`、`适应窗口`、`显示全部`、`重置`。
- 增加 `日 / 周 / 月 / 总览` 分段控件。
- 保留关系线显示开关。
- 保留主线创建、刷新。
- 工具栏视觉向概念图靠近：紧凑、分组、低噪声。

### 2. 时间轴模式

- 新增 `TimeAxisMode = day | week | month | overview`。
- 日视图保留连续日期。
- 周视图按周降低 tick 密度。
- 月视图弱化每日 tick，强化月内关键刻度。
- 总览优先显示有任务日期、今天和任务范围端点。

### 3. 主线显示控制

- 主线右键支持折叠/展开。
- 主线右键支持隐藏/显示。
- 主线右键支持固定/取消固定。
- 工具栏支持管理隐藏主线和显示全部主线。
- 隐藏主线不会修改任务文件。
- 折叠主线不删除任务，只压缩显示。

### 4. 任务密集聚合

- 同一日期同一主线任务超过阈值时进入紧凑显示。
- 第一版显示前几个任务，并以 `+N` 聚合节点展开剩余任务。
- 聚合展开状态是视图状态，不写入任务 md。

### 5. 视图状态持久化

- 持久化 `panX`、`panY`、`canvasZoom`、`timeScale`、`centerDate`。
- 持久化 `timeAxisMode`、`showRelations`、`showHiddenMainlines`。
- 持久化 `laneZooms` 和 `expandedClusters`。
- 写入插件 settings，不写入任务 frontmatter。

### 6. 概念图视觉靠近

- 深色画布层次更清晰。
- 顶部工具栏控件分组。
- 日期 tick 有当前日期、中心日期和主要刻度状态。
- 主线标签显示任务数量、固定/折叠/隐藏状态。
- 主线脊骨颜色更明显。
- 任务节点更紧凑，状态和优先级更清楚。
- relation 默认弱化，hover 时突出。

### 7. 快捷键

- `T`：今天。
- `F`：适应窗口。
- `R`：显示/隐藏 relation。
- `0`：重置视图。
- `1/2/3/4`：日/周/月/总览。

快捷键只在鱼骨画布聚焦时生效，输入框内不触发。

## 不做项

- 不做右侧工作台。
- 不做快速笔记 / Codex 输入。
- 不做每日总结。
- 不做 relation 编辑器。
- 不做拖拽创建 relation。
- 不扫描真实 Vault 旧笔记。
- 不修改标准任务 schema。
- 不创建默认主线。

## 手动验收

测试 Vault：

```text
E:\主线规划\主线规划
```

1. 打开鱼骨画布视图。
2. 切换日/周/月/总览，确认日期轴变化且任务仍对齐。
3. 点击今天、跳转、适应窗口、显示全部，确认视图移动合理。
4. 右键主线折叠、隐藏、固定，确认任务文件不被批量修改。
5. 显示隐藏主线，确认隐藏主线可管理。
6. 创建同日同主线多个任务，确认出现 `+N` 聚合节点。
7. 展开聚合节点，确认隐藏任务出现。
8. hover 任务和 relation，确认高亮仍有效。
9. 重启或刷新 Obsidian，确认视图模式和缩放状态保留。
10. 检查整体 UI 是否更接近概念图：紧凑工具栏、清晰日期轴、明显主线标签、胶囊式任务节点。
