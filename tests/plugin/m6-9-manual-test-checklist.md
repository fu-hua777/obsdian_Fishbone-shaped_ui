# M6.9 模块管理验收清单

## 前置条件

- 已启用 Fishbone Planner 插件。
- 鱼骨画布视图右侧工作台可见。

## 验收步骤

1. 点击右上角“模块管理”，确认弹窗列出所有右侧模块：
   - 进度概览
   - 今日聚焦
   - 本周重点
   - 主线进度
   - 每日总结
   - 时间 / 天气
2. 在弹窗中关闭一个模块显示并保存，确认右侧面板立即隐藏该模块。
3. 重新打开“模块管理”，确认每个模块只有显示开关，没有第二列折叠选项；打开该模块显示并保存，确认模块恢复。
4. 在模块标题栏点击“折叠”，确认模块只保留标题栏；点击“展开”后内容恢复。
5. 在模块标题栏点击“隐藏”，确认模块隐藏，并可通过“模块管理”恢复。
6. 拖动模块标题调整顺序、拖动模块底部调整高度，确认仍然可用。
7. 点击“恢复默认布局”，确认模块顺序、高度、显隐、折叠状态回到默认。
8. 重载 Obsidian，确认模块顺序、高度、显隐和折叠状态保留。

## 通过标准

- 所有管理动作都写入 `dashboardState`，刷新后保留。
- 隐藏所有模块时右侧面板给出恢复提示，不报错。
- M6.5 的 L 型工作台、M6.7 每日总结和 M6.8 时间天气模块不被破坏。
- `npm run typecheck`、`npm run build` 和 `node scripts/validate-m6-9-module-management.js` 通过。
