# M6.1-M6.2 手动验收清单

## 前置

- Obsidian 已启用 Fishbone Planner 插件。
- 打开 `Fishbone Planner 鱼骨时间视图`。
- Vault 中存在若干 `PlanningSystem/Tasks` 标准任务。

## M6.1 数据聚合

- 右侧辅助面板中的任务数量与画布摘要任务数量一致。
- 今日聚焦只显示今天日期的任务。
- 本周重点显示本周一到周日范围内的高优先、阻塞或进行中任务。
- 进度概览按用户主线显示完成率。
- 未分配任务显示为 `未分配`，不会自动创建默认主线。

## M6.2 右侧面板

- 右侧辅助区按功能拆成独立模块，而不是一个整体大卡片。
- 辅助区整体高度与当前显示页面匹配，默认不出现外层纵向滚动条。
- 今日聚焦、本周重点、主线进度等内容过多时，只在对应小模块内部滚动。
- 不再显示 `状态速览` 模块，避免和后续底部待办/进行中/已完成模块重复。
- 拖动任意模块标题，可以调整右侧模块顺序。
- 点击模块标题右侧的 `宽` / `窄` 按钮，可以切换该模块占用宽度。
- 刷新鱼骨视图后，模块顺序和宽度保持。
- 点击工具栏 `隐藏辅助` 后，右侧辅助面板消失，鱼骨画布仍可拖动和缩放。
- 点击 `显示辅助` 后，右侧辅助面板恢复。
- 拖动辅助面板左侧分隔条可以调整面板宽度。
- 刷新鱼骨视图后，辅助面板显示状态和宽度保持。
- 窄窗口下右侧面板不会覆盖画布顶部工具栏。

## 本阶段不验收

- 不测试快速输入写入任务。
- 不测试每日总结生成。
- 不测试状态看板拖拽。
