# M5.1 鱼骨画布手动验证清单

测试 Vault：

```text
E:\主线规划\主线规划
```

## 前置条件

- `npm run build` 已通过。
- `node scripts/install-m3-plugin.js` 已执行。
- Obsidian 已重载。
- `Fishbone Planner` 插件已启用。

## 验证步骤

1. 打开命令面板。
2. 运行 `Fishbone Planner: 打开鱼骨时间视图`。
3. 确认主图看起来是画布，而不是表格列/行。
4. 按住画布空白处拖动，确认日期轴、主线和任务一起平移。
5. 在主线外按住 `Ctrl` 并滚动鼠标滚轮，确认整体画布缩放。
6. 把鼠标放在某条主线区域内，按住 `Ctrl` 并滚动，确认只有该主线高度和分支距离变化。
7. 把鼠标放在日期轴上按住 `Ctrl` 并滚动，确认日期间距变化。
8. 按住 `Alt` 并滚动，确认也能调整日期间距。
9. 点击 `重置视图`，确认画布回到默认视图。
10. 点击任务节点，确认仍能打开任务 md。
11. 勾选/取消勾选任务，确认状态仍能更新。
12. 修改任务主线选择，确认任务仍能分配到主线。
13. 新建、修改、删除、排序主线，确认 M5 功能不回退。

## 不应出现

- 不应自动创建默认主线。
- 不应覆盖已有 `PlanningSystem`。
- 不应把主图继续渲染成表格。
- 不应在点击 checkbox、select、任务节点时拖动画布。
