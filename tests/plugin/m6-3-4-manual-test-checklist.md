# M6.3-M6.4 手动验收清单

## 前置

- 已打开 `Fishbone Planner 鱼骨画布视图`。
- 右侧辅助面板处于显示状态。
- Vault 中存在今日任务、本周任务、阻塞任务、进行中任务和至少一条用户主线。

## M6.3 今日聚焦与状态看板

- 今日聚焦任务行左侧显示完成框。
- 勾选今日任务后，任务 md frontmatter 的 `status` 写为 `done`，鱼骨画布刷新。
- 取消勾选已完成任务后，任务 md frontmatter 的 `status` 写为 `todo`，鱼骨画布刷新。
- 今日聚焦任务行的状态下拉可以切换 `todo / doing / done / blocked / canceled / inbox`。
- 状态看板按待办、进行中、阻塞、已完成分列显示代表任务。
- 在状态看板中切换任务状态后，对应任务会移动到新的状态分组。
- 点击任务主体仍能打开对应任务 md，点击完成框或状态下拉不会误打开任务。

## M6.4 本周重点与进度概览

- 本周重点显示高优先、阻塞、进行中、过期任务，并且同一任务不会重复出现。
- 本周重点任务显示原因标签，例如 `过期`、`高优先`、`阻塞`、`进行中`。
- 主线进度模块显示每条主线的完成率和 `总 / 进 / 阻 / 高 / 期` 指标。
- 未分配任务仍显示在 `未分配` 组，不创建默认主线。

- 不测试快速输入候选任务。
- 不测试每日总结生成。
